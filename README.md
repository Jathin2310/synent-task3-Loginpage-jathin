# synent-task3-Loginpage-jathin
A responsive and modern login UI featuring email and password authentication fields, remember me functionality, forgot password option and a clean user centric design.

# Responsive Login UI

## Technologies Used
- HTML5
- CSS3
- Visual Studio Code
- GitHub

## Features
- Responsive design
- Email input field
- Password input field
- Remember Me option
- Forgot Password link
- Sign Up navigation link
- Modern and clean user interface
- Mobile-friendly layout

## Project Structure

```text
LoginUI/
│
├── index.html
├── style.css
└── README.md
```

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. The Login UI page will be displayed.
